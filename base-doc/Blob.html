Blob

Binary blobs

type Blob = Prim.Types.Blob

An immutable, possibly empty sequence of bytes. Given b : Blob:

  • b.size() : Nat returns the number of bytes in the blob;
  • b.vals() : Iter.Iter<Nat8> returns an iterator to enumerate the bytes of the blob.

(Direct indexing of Blobs is not yet supported.)

public let hash : (b : Blob) -> Nat32

Returns a (non-cryptographic) hash of 'b'

public func equal(x : Blob, y : Blob) : Bool

Returns x == y.

public func notEqual(x : Blob, y : Blob) : Bool

Returns x != y.

public func less(x : Blob, y : Blob) : Bool

Returns x < y.

public func lessOrEqual(x : Blob, y : Blob) : Bool

Returns x <= y.

public func greater(x : Blob, y : Blob) : Bool

Returns x > y.

Returns x >= y.

public func compare(x : Blob, y : Blob) : {#less; #equal; #greater}

Returns the order of x and y.
